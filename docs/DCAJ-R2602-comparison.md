## 改善サマリー

| 指標 | 改善前 | 改善後 |
|:-----|:-------|:-------|
| textarea（自由記述） | 7問 | **2問** (Q1a, Q5a) |
| multi_choice（選択式） | 3問 | **10問** |
| 「その他（自由記述）」付き | 0問 | **7問** |
| 回答時間の目安 | 自由記述が多く個人差大 | 選択式中心で安定的に5分以内 |
| 分析容易性 | 低（テキスト分類が必要） | **高**（定量分析直結） |
